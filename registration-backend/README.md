# Patient & Owner Registration Backend

This directory contains the backend implementation for patient and owner registration functionality.

## Features
- Patient and Owner registration
- Data validation
- Duplicate prevention
- Authentication and security
- Database storage
- Audit logging

## Structure
- `models/` - Database models
- `routes/` - API endpoints
- `controllers/` - Business logic
- `middleware/` - Validation and security middleware
- `services/` - Data access services
- `config/` - Configuration files
